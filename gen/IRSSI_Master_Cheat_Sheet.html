IRSSI Master Cheat Sheet

Connecting and Managing Networks

  • /connect - Connect to a server
  • /disconnect - Disconnect from a server
  • /server list - List all servers

Joining and Managing Channels

  • /join <#channel> - Join a channel
  • /part <#channel> - Leave a channel
  • /channel list - List all channels

Sending Messages

  • /msg - Send a private message
  • /say - Send a message to the current channel

Managing Nicknames

  • /nick - Change your nickname
  • /whois - Get information about a user

Other Useful Commands

  • /away - Set away status
  • /quit - Quit the IRC client
  • /ignore - Ignore a user
  • /unignore - Unignore a user
  • /theme list - List available themes
  • /load - Load a theme

Window Management

  • /window new - Open a new window
  • /window close - Close the current window
  • /window goto - Go to a specific window
  • /window list - List all windows

Advanced Commands

  • /whois host - Check the IP address and hostname of a user
  • /dns - Resolve the IP address for a nickname
  • /stats u - Get server uptime
  • /stats c - Get information about server connections
  • /trace - Trace information about a nickname or server
  • /links - Get a list of all linked servers
  • /list *a* - List all channels containing the letter ‘a’
  • /lusers - Get information about the number of users and servers
  • /whowas - Find information about a user who is no longer connected
  • /admin - Get admin info of the server
